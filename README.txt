Veuillez suiver les instructions suivantes :

1.  Ouvrez le terminal dans le fichier contenant le code source

2.  Tapez make dans le terminal

3.  Tapez ensuite ./blockchain_network

4.  Un menu doit s'afficher, entrer un nombre entre 1 et 5 selon ce que vous voulez faire

Dans graph.txt on a :

le nombre de noeud
le nombre d'aretes
noeud1 noeud2 
noeud2 noeud1
etc ....

A bien noter que on les noeuds communique dans les deux sens !
